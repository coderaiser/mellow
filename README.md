Mellow - convert path from web to windows.

## Install

```
npm i mellow
```

## API

### webToWin

Previouse has name `pathToWin`.

```js
const mellow  = require('mellow');
mellow.webToWin('/c/windows');
// returns
'c:\\windows';
```

### winToWeb

Previouse has name `pathFromWin`.

```js
const mellow  = require('mellow');
mellow.winToWeb('c:\\windows');
// returns
'/c/windows';
```
